## Login / Sign Up
1. Go to [SubVerse Dashboard](https://dashboard.subverseai.com/)
2. Sign up or log in with your credentials.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/workflow/quickstart/create-workflow.png"
/>

## Create a New Workflow
1. Navigate to the Workflow section.
2. Click **Create New Workflow** and give it a meaningful name.

## Edit & Define the Workflow
1. Add data source blocks to push data from your databases and trigger points.
2. Connect agent blocks to process this data using AI.
3. Use function blocks to trigger external actions (e.g., sending emails, making API calls).

## Activate the Workflow
1. Once all blocks are configured, save & activate the workflow.
2. The workflow will now run automatically based on predefined triggers.
