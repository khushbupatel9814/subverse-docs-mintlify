## Login / Sign Up
1. Go to [SubVerse Dashboard](https://dashboard.subverseai.com/)
2. Sign up or log in with your credentials.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/voice-agents/quickstart/voice-agents-list.png"
/>

## Create a Voice Agent
1. Click on **Create New Agent** or select a template from the list.
2. For starters, you can select from the default pre and post sales (inbound and outbound) use cases from templates.
3. To create a new use case, provide a **Usecase Name** (only alphanumeric and underscores are allowed).

<Tip>
  To save time, you can also duplicate any existing use case by clicking on the copy icon.
</Tip>

## Customize your Agent
- **Initial Prompt:** Set an introductory message for the agent.
- **System Prompt:** Define agent behavior, identity, and response style.
- **Speech Detection Settings:** Choose language, interruptions, and accuracy.
- **Voice Settings:** Select TTS language, voice, quality, and speed.
- **Call Settings:** Configure transfer numbers (for realtime escalations to human agents), communication methods, and DTMF settings.
- **LLM Tools:** Select tasks like `end_call`, `transfer_call`, or `query_knowledge_base`.
- **Knowledge Base:** Attach relevant company documents for agent reference.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/voice-agents/quickstart/edit-voice-agent.png"
/>

## Test Call
1. Once you save the agent configuration, hover over to the voice agent you just created.
2. Click on **Make Call** icon to initiate a test conversation.
3. Adjust settings as needed and refine the agent responses.

## Setup a Phone Number
1. Visit **Telephony** tab, you will see Outbound and Inbound numbers sections.
2. Note down the outbound phone number assigned to you.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/integrations/telephony/telephony-dashboard.png"
/>

## Get an API Key
1. Visit **API keys** tab, and create a new API key.
2. Copy and save the generated API key securely.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/integrations/api-keys/api-keys.png"
/>
